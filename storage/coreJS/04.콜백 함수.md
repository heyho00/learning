# 콜백 함수

## 콜백 함수는 함수다.

```js
var obj = {
  vals: [1, 2, 3],
  logValues: function (v, i) {
    console.log(this, v, i);
  },
};

obj.logValues(1, 2); //{vals: [1,2,3], logValues: f} 1 2
[4, 5, 6].forEach(obj.logValues); //window {...} 4 0 //window {...} 5 1 //window {...} 6 2
```

forEach에 의해 콜백이 함수로서 호출되어 전역 객체를 바라보게 된다.
